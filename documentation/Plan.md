### План тестирования мобильного приложения "Мобильный хоспис".

#### Объект тестирования Приложение “Мобильный хоспис” на системе Android. Версия 1.0.0.

Приложение дает функционал по работе с новостями хосписа и включает в себя:

- информацию о новостях и функционал для работы с ними;
- тематические цитаты;
- информацию о приложении.

#### Цели тестирования

- Проверка функциональности приложения и пользовательского интерфейса (составление чек-листа и тест-кейсов, проведение ручного и автоматизированного тестирования, составление баг-репортов и формирование отчетности)

#### Этапы тестирования

- Определение типов тестирования
- Ручное тестирование приложения
- Составление чек-листа и написание тест-кейсов
- Написание и проведение UI-тестов
- Подготовка отчета о проведенном тестировании

##### Виды тестирования

- Исследовательское тестирование - изучаем работу приложения с целью определения его границ
- Тестирование установки - проверяем установку и удаление приложения на устройстве.
- Функциональное тестирование - проверяем работоспособность всех кнопок и форм приложения, корректность перехода по ссылкам.
- Тестирование пользовательского интерфейса (UI Testing) - проверяем удобство интерфейса приложения, а так же корректность отображения его элементов (наличие визуальных дефектов, удобство кнопок, размер шрифтов, адаптация к разным экранам )
- Тестирование на удобство использования (Usability Testing) - проверяем приложение на соответствие потребностям пользователей.
- Позитивное тестирование - проверяем работу приложения при вводе корректных данных
- Негативное тестирование - проверяем корректность отработки приложения при вводе неверных данных

##### Границы тестирования приложения

- Экран загрузки
  - Наличие экрана загрузки и спиннера
- Форма авторизации
  - Авторизация с валидным логином и паролем
  - Отказ в авторизации с пустыми полями
  - Отказ в авторизации незарегистрированному пользователю
  - Отказ в авторизации с неверным логином или паролем
  - Отказ в авторизации с пробелами в поле логина или пароля
- Панель навигации приложения
  - Menu
    - Переход между страницами Main, News, About через кнопку Menu
  - Цитаты
    - Проверка видимости цитат
    - Развернуть-свернуть цитату
  - Log out (выход из приложения)
- Страница Main
  - Развернуть-свернуть новости
  - Переход на страницу News через All News
- Страница News
  - News
    - Просмотр новостей
    - Сортировка новостей по дате добавления
    - Фильтрация новостей
      - по категории
      - по дате /без даты /с только датой от/ с только датой до
      - отмена внесенного фильтра по кнопке Cancel
  - Control panel
    - Просмотр новостей
    - Сортировка новостей по дате добавления
    - Фильтрация новостей
      - по категории
      - по дате /без даты /с только датой от/ с только датой до
      - по статусу Active/ Not active
      - отмена внесенного фильтра по кнопке Cancel
    - Создание новости с валидными данными
    - Отказ в создании новости
      - с пустыми полями
      - без категории
      - со своей категорией
      - без титула
      - без даты и времени публикации
      - без текста новости
      - отмена создания новости по Cancel
    - Редактирование новости
      - изменить категорию на доступную из списка/на свою категорию
      - изменить титул
      - изменить дату и время публикации(текущая дата/ будущая дата)
      - отказ в изменении даты на прошедшу.
      - изменить текст публикации
      - отмена внесенных изменений по кнопке Cancel
- Страница About
  - Возврат на предыдущую страницу через кнопку назад
  - Переход по ссылке на соглашение о конфиденциальности информации
  - Переход по ссылке на пользовательское соглашение

- Зависимость работы приложения от устройства
  - Установка приложения на разных устройствах
  - Удаление приложения
  - Поведение приложения при
    - входящем звонке
    - СМС/push уведомлении
    - изменении положения экрана
    - При нажатии кнопки назад на устройстве
    - при блокировке /разблокировке телефона
    - при включении режима авиа
  - Интернет
    - без интернета
    - при подключении к Wi-Fi/ 2G/3G/4G
    - при переключении между сетями
  - Поведение приложения при ограниченном количестве памяти
  - при низком заряде батареи
  - при зарядке телефона
  - при смены темы телефона на темную

---

Не автоматизированы:

- тест-кейсы установки, уделаения приложения (Test Case с 1 по 4)
- тест-кейсты на удобство использования приложения, а так же зависимости работы приложения от характеристик устройства (Test Case с 74 по 89)
- тест-кейсы на сортировку новостей (Test Case 35 и 44)
- тест-кейс на изменение даты публикации новости на прошедшую дату.

Отсутствие автоматизации связано со сложностью в реализации автотестов.
